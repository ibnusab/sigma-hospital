# 🏥 Sigma Hospital

### Smart Hospital Management System for Android

Modern Android Hospital Management Application built with **Kotlin**, **MVVM Architecture**, **Room Database**, and **Google AI** for preliminary diagnosis assistance.

![Android](https://img.shields.io/badge/Android-API%2024+-3DDC84?style=for-the-badge&logo=android)
![Kotlin](https://img.shields.io/badge/Kotlin-2.x-7F52FF?style=for-the-badge&logo=kotlin)
![Material3](https://img.shields.io/badge/Material-Design%203-2196F3?style=for-the-badge)
![Room](https://img.shields.io/badge/Room-Database-1976D2?style=for-the-badge)
![MVVM](https://img.shields.io/badge/Architecture-MVVM-success?style=for-the-badge)

---

A professional Android application designed to help hospitals manage patients, doctors, appointments, medical records, and AI-assisted diagnosis in one integrated platform.

# 🏗 Architecture

Project uses **MVVM (Model View ViewModel)** architecture.

```
UI
│
├── ViewModel
│
├── Repository
│
├── Room Database
│
└── Google AI
```

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/ibnusab/sigma-hospital.git
```

Open project

```
Android Studio
```

Sync Gradle

Run Application

```
Shift + F10
```
